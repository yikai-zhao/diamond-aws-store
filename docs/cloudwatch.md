# AWS CloudWatch Monitoring

- **Metrics**: 
  - EC2 CPU Utilization
  - RDS Connections
  - Request Count
  - Error Rate

- **Logs**: 
  - Application logs streamed via CloudWatch Agent
  - Docker logs forwarded to CloudWatch

- **Alerts**:
  - High CPU (>80%)
  - High latency (>200ms)
  - Error rate > 5%
